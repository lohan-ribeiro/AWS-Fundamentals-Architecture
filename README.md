# Diagrama de Arquitetura AWS – Gerenciamento de Arquivos e Backup
Projeto desenvolvido para aplicar os conceitos fundamentais da AWS, incluindo Amazon S3, AWS Lambda, Amazon S3 Glacier, Amazon EC2, Amazon EBS, Snapshots, AMIs e Amazon RDS por meio da construção de uma arquitetura de armazenamento, backup e gerenciamento de arquivos na nuvem.
